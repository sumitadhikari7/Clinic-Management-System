# Rural Clinic Medical Supply & Patient Visit Tracker (HealthTrack)

A PHP + MySQL desktop application for managing patient visits, appointments, medical supplies, staff activity, and reports.

> **Note:** Desktop-only. Mobile responsiveness will be added later.

## Features
- Patient visit tracking
- Appointment scheduling and editing
- Medical supply inventory management
- Admin/staff login (MD5 hashed passwords)
- Dashboard with live statistics
- Report generation

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: Apache (XAMPP)
